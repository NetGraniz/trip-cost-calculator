# Калькулятор стоимости поездки

Одностраничный калькулятор для расчета топлива, стоимости топлива, платных дорог и общей стоимости поездки.

## Открыть сайт

https://netgraniz.github.io/trip-cost-calculator/

## Возможности

- Расчет топлива по расстоянию и расходу
- Расчет общей стоимости поездки
- Учет платных дорог
- Режим поездки туда и обратно
- Светлая и темная тема
- Сохранение последних значений в браузере
- Адаптивный дизайн для телефона и компьютера
